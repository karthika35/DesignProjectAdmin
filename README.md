# DesignProjectAdmin
angular 6 website which can use to upload advertisements using firebase database

## installation
windows 10 os <br>
install node js<br>
install angular 6<br>

 ```bash
   git clone https://github.com/karthika35/DesignProjectAdmin.git
   npm i
```

## run
for browser 
   ```bash
   ng serve
 ```
